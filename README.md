## モデリング（Blender）

本プロジェクトでは、Blenderを使用して3Dモデルの作成を担当しました。

Unity内で使用するオブジェクトのモデリングを行い、ゲーム内に組み込んでいます。

また、モデルの配置やスケール調整も行い、ゲームの見た目を整えるようにしました。
- 教室内のオブジェクトをモデリング
- 小道具（机・壁など）を作成
## プロジェクト概要

本プロジェクトは、学園祭にて展示したVR音楽ゲームです。

プレイヤーが音楽に合わせて操作を行う体験型ゲームとして制作し、実際に来場者にプレイしてもらうことを目的として開発しました。

VR環境での没入感を意識し、視覚的な表現や操作性を重視して設計しています。
## 担当内容

- Blenderで3Dモデル作成
- Unity内へのモデル組み込み
- 配置・スケール調整
## 工夫した点

大学の教室をモデルに、実際の空間をほぼ100%再現することを目指して3Dモデリングを行いました。

机や壁などの配置やスケールを現実に近づけることで、VR空間における没入感を高める工夫をしています。

実際の教室に近い環境を再現することで、プレイヤーが直感的に空間を理解できるよう設計しました。
