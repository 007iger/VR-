## モデリング（Blender）

本プロジェクトでは、Blenderを使用して3Dモデルの作成を担当しました。

Unity内で使用するオブジェクトのモデリングを行い、ゲーム内に組み込んでいます。

また、モデルの配置やスケール調整も行い、ゲームの見た目を整えるようにしました。
- 教室内のオブジェクトをモデリング
- 小道具（机・壁など）を作成
